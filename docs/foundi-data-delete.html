1. 데이터 삭제 요청 방법

파운드아이(Found-i) 사용자는 아래의 방법으로 본인의 데이터 삭제를 요청할 수 있습니다.

방법 1: 앱 내에서 직접 삭제

1 파운드아이 앱을 실행합니다.
2 마이페이지로 이동합니다.
3 데이터 삭제(탈퇴) 버튼을 선택합니다.
4 안내 사항을 확인하고 삭제를 진행합니다.

방법 2: 이메일을 통한 삭제 요청

앱을 삭제했거나 앱에 접근할 수 없는 경우, 아래 이메일로 데이터 삭제를 요청할 수 있습니다.

2. 삭제되는 데이터

삭제 요청 시 다음 데이터가 삭제됩니다.

데이터 유형 삭제 여부 비고
기기 식별자(익명 ID) 즉시 삭제 사용자 계정 완전 삭제
등록한 분실물 정보 즉시 삭제 제목, 설명, 위치 정보 포함
업로드한 이미지 즉시 삭제 Cloudflare R2에서 완전 삭제
아파트 단지 설정 정보 즉시 삭제 -
푸시 알림 토큰 즉시 삭제 알림 수신 중단
서비스 이용 로그 30일 후 삭제 법적 의무 보관 기간 준수

주의: 데이터 삭제 후에는 복구할 수 없으며, 등록하신 분실물 정보도 모두 삭제됩니다.

3. 추가 보관이 필요한 데이터

다음의 경우 관련 법령에 따라 일정 기간 보관 후 삭제합니다.

  • 서비스 이용 기록: 삭제 요청일로부터 30일간 보관 후 파기 (통신비밀보호법)

위 보관 기간이 경과하면 해당 데이터는 지체 없이 파기됩니다.
