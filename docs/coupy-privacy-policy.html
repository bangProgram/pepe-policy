1. 수집하는 개인정보 항목 및 수집 방법

앱은 서비스 제공을 위해 다음과 같은 정보를 수집합니다.

수집 항목 수집 방법 저장 위치 수집 목적
쿠폰 이미지 사용자가 직접 선택하여 등록 기기 내부 저장소 쿠폰 관리 및 정보 추출

2. 개인정보의 이용 목적

수집된 정보는 다음의 목적으로만 이용됩니다.

  • 쿠폰 관리 서비스 제공: 쿠폰 등록, 저장, 분류, 조회, 바코드 제시
  • 쿠폰 정보 자동 추출: 이미지에서 브랜드, 상품명, 유효기간 등을 인식
  • 만료 알림: 유효기간 임박 시 로컬 알림 제공
  • 광고 제공: Google AdMob을 통한 광고 표시

3. 개인정보의 제3자 제공

앱은 서비스 제공을 위해 다음의 제3자에게 최소한의 정보를 제공합니다.

제공받는 자 제공 항목 제공 목적 보유 기간
Anthropic (Claude API) 쿠폰 이미지 쿠폰 정보 자동 추출 (브랜드, 상품명, 유효기간, 바코드 등) API 처리 완료 후 즉시 파기 (서버 미보관)
Google (AdMob) 광고 식별자, 기기 정보 맞춤형 광고 제공 Google 개인정보처리방침에 따름
Google ML Kit (바코드 인식): 모든 처리가 사용자의 기기에서 수행되며, 이미지나 바코드 데이터가 외부 서버로 전송되지 않습니다.

4. 개인정보의 보관 및 파기

4-1. 기기 내 데이터 보관

모든 쿠폰 데이터(이미지, 바코드, 쿠폰 정보)는 사용자의 기기 내부에만 저장되며, 외부 서버에 별도 보관하지 않습니다.

4-2. 자동 정리 정책

  • 사용 완료 쿠폰 이미지: 사용 완료 후 30일 경과 시 자동 삭제
  • 사용 완료 쿠폰 기록: 사용 완료 후 60일 경과 시 자동 삭제
  • 보호 설정 쿠폰: 사용자가 보호(잠금) 설정한 쿠폰은 자동 삭제에서 제외

4-3. 앱 삭제 시

앱을 삭제하면 기기에 저장된 모든 쿠폰 데이터(이미지, 데이터베이스)가 완전히 삭제됩니다.

5. 이용자의 권리와 행사 방법

사용자는 다음과 같은 권리를 행사할 수 있습니다.

  • 데이터 삭제: 앱 내에서 개별 쿠폰을 언제든 직접 삭제할 수 있습니다.
  • 전체 데이터 삭제: 앱을 삭제하면 모든 데이터가 완전히 제거됩니다.
  • 광고 식별자 관리: Android 설정 → Google → 광고에서 광고 ID를 재설정하거나 맞춤 광고를 비활성화할 수 있습니다.
  • 카메라/저장소 권한: Android 설정에서 앱의 카메라, 저장소 접근 권한을 언제든 철회할 수 있습니다.

6. 앱 접근 권한

앱은 서비스 제공을 위해 다음의 기기 접근 권한을 요청합니다.

권한 목적 필수 여부
카메라 (CAMERA) 쿠폰 이미지 촬영 및 바코드 스캔 선택
저장소 (READ_EXTERNAL_STORAGE, READ_MEDIA_IMAGES) 갤러리에서 쿠폰 이미지 선택 선택
인터넷 (INTERNET) Claude API 호출, 광고 로드 필수

선택 권한은 거부하더라도 해당 기능 외의 서비스는 정상적으로 이용할 수 있습니다.

7. 아동의 개인정보 보호

앱은 만 14세 미만 아동을 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다. 만 14세 미만 이용자의 개인정보가 수집된 사실을 인지할 경우, 해당 정보를 즉시 파기하겠습니다.

8. 개인정보의 안전성 확보 조치

  • 데이터 암호화: 외부 서버와의 통신은 HTTPS(TLS)를 통해 암호화됩니다.
  • 로컬 저장: 쿠폰 데이터는 앱 전용 내부 저장소에 저장되어 다른 앱에서 접근할 수 없습니다.
  • 최소 수집 원칙: 서비스에 필요한 최소한의 정보만 수집합니다.

9. 개인정보처리방침의 변경
